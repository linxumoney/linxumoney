## 💡 这个计划是什么

我在用 AI 做内容创作（公众号、YouTube、X）的过程中，沉淀了一套工具链。

这些工具帮我做到：
- 每天 30 秒选出今天该写什么
- 一句话生成完整公众号文章并推送草稿

**我决定把它们全部开源。**
